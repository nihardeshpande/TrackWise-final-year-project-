# TrackWise-final-year-project-
Project Title: Productivity Tracking and Analytics System (TRACKWISE)

Introduction: The Productivity Tracking and Analytics System is a software application designed to help users measure, analyze, and improve their productivity. The system focuses on tracking task completion, goal progress, and time spent on work using tools such as Pomodoro sessions, Focus sessions and analytics dashboards. By providing visual insights and performance statistics, the system enables users to manage their time efficiently and achieve their goals effectively.

System Overview: The system allows users to sign in securely and access a personalized productivity environment. Once logged in, users can View their productivity summary on the dashboard, Create and manage my tasks & my goals lists, Define and track long-term goals, Measure focused work sessions using Pomodoro or Stopwatch, Analyze productivity using daily, weekly, and monthly analytics, View tasks and activities using a calendar, Manage personal information through settings, The system actively measures task completion and time usage to generate meaningful insights.

Functional Description:

3.1 User Authentication: Users must sign in using valid credentials. Authentication ensures data security and personalized access. Each user’s productivity data is stored separately.

3.2 Dashboard: The dashboard provides a quick overview of user productivity. Displays: Total tasks completed, Ongoing tasks, Active goals, Total productivity time, Acts as the central control panel of the system. Also dashboard provides access to all the pages.

3.3 My-tasks List Management: Users can create, update, and delete My-tasks. Tasks include Task name, Description, Priority Status (Completed / Pending). These tasks are independent of goal-based tasks. In this page,User can also start a pomodoro or focus session related to a specific task.

3.4 Goal Setting and Goal-Based Tasks: Users can set specific goals (e.g., “Complete Semester Project”). Each goal contains a separate list of tasks(steps) required to complete that goal. The system tracks Total tasks under a goal, Completed tasks, Remaining tasks, Goal tasks are different from normal to-do list tasks. Goal progress is shown as a percentage or progress bar.

3.5 Pomodoro / Stopwatch Session: Users can start a Pomodoro session (focused work + break cycles). A stopwatch mode is available for flexible time tracking. The system records: Session duration, Associated task or goal (optional). This helps measure productive work time accurately.

3.6 Analytics Dashboard: Analytics Dashboard provides detailed productivity insights based on tasks, goals,total pomodoro sessions and total focus session time. Analytics are displayed in: Daily, Weekly, Monthly formats. It shows: Total tasks completed, Time spent in focus sessions, Goal progress analysis, Remaining tasks. Helps users to identify productivity patterns and improvement areas.

3.7 Calendar View(optional): Calendar View displays tasks, goals, and sessions on a calendar interface. Users can: View tasks by date, Track deadlines, Review past productivity Enhances, planning and scheduling efficiency.

3.8 Settings: Users can manage Personal details (name, email, profile), Preferences (time format, notifications), Ensures customized user experience.

Key Features: Summary Secure user authentication, Dashboard overview, To-do list management, Goal creation, tracking Pomodoro and stopwatch sessions, Advanced analytics (daily, weekly, monthly), Calendar integration, User settings management.

Conclusion: The Productivity Tracking and Analytics System provides a complete productivity management solution by combining task tracking, goal management, time measurement and analytics. The system helps users stay organized, improve focus, and achieve their goals through data-driven insights and intuitive features.
